Custom Product Manager

Custom Product Manager is a lightweight WordPress plugin that allows you to create, manage, and display products without using WooCommerce. It is ideal for small businesses and simple product listings where a full eCommerce system is not required.

Features
Add and manage products from WordPress admin
Product title, image, and description support
Lightweight and fast performance
Clean and responsive product layout
Easy to integrate using shortcode
Works with any WordPress theme

Installation
Download or clone this repository
Upload the custom-product-manager folder to /wp-content/plugins/
Activate the plugin from the WordPress Admin Dashboard
Use the shortcode to display products

Usage
Use the following shortcode to display products:
[custom_product_manager]

Requirements
WordPress 5.0 or higher
PHP 7.0 or higher
